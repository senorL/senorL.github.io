hexo d命令错误的解决方法

senor
2022-08-31

方法一

确认hexo-deployer-git的模块是否下载成功

复制下面这行代码到命令行。

npm install hexo-deployer-git --save

方法二

修改 _config.yml文件

注意 我使用win11系统打开该文件时，在type后面有单引号，应将单引号删去，也可以直接复制下面的代码。
1.将GitHub配置地址的方式从http改为ssh
2.将branch后的master换成main

方法三

在命令行中写入你的github名称与注册邮箱
