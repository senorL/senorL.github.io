数据结构

内存

变量 ：先定义再使用

（变量类型）（变量标识符）
int max
int max = 0 赋予初始值

整数型

基本型 int
短整型 short
长整型 long
区别：内存空间不同
sizeof 用于计算某种类型的对象在内存中所占的字节数
int 默认为 signed int 即为有符号
0x 后是十六进制数 0 后是八进制数

浮点型

浮点型=实型

浮点型 有效值
float 7位
double 15位
long double 15位

字符型

1.一个字符型占一个字节
2.存储类型与整型相同
3.可以与整型数据相互赋值
4.可以和整型一样进行运算

  • 转义字符

布尔型

存储“真” “假” 的变量
占一个字节
值只能为1或0

  • 赋0 为0
  • 赋非0 为1

变量命名

标识符只能由字母、数字和下划线三种字符组成，且第一个字符必须为字母或者下划线，不可以与保留字相同

常量

在程序运行过程中，其值保持不变的量
字面常量： 1 、2、4
符号常量：用一个标识符表示一个常量
类型前加上关键词 const
好处： 不会被修改

常量的类型用后缀分辨

关于编程题的一些笔记

在C++中可以#include ，然后cout << fixed << setprecision(n）来控制精度
其中fixed是小数点的意思，setprecision是设置精度，括号中的数表示保留几位精度输出
